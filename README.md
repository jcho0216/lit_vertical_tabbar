## Usage
```
LitVerticalTabBar(
  backgroundColor: Colors.black87,
  padding: const EdgeInsets.symmetric(vertical: 24),
  moveTabToLastIndexOnScrollEnd: true,
  children: [LitElementModel(tabName: "tab1", child: ChildWidget())],
  divider: Divider(),
)
```
